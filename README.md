 Sobre o projeto
</h2>

<p>
Lista de presença para eventos e outras utilidades, basta adaptá-la.
</p>

<p>
Os usuários podem registrar sua presença colocando apenas o seu nome.
</p>
<p>
Após o registo deles, é fornecido o nome do usuário e o horário exato do registro da presença. 
</p>
<p>
Ao adicionar um novo usuário, o estado do Input no campo de texto é atualizado automaticamente.
</p>

<p>
 Além disso, o usuário poderá excluir alguma despesa já paga ou indesejada.
</p>

<p>
 Esse projeto foi desenvolvido durante o curso discover da Rocketseat.
</p>
